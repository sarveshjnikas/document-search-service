This prototype uses:
- Express API
- MySQL as the system of record
- Elasticsearch as the search engine
- Redis as the search cache
- In-memory tenant rate limiting
- Direct synchronous indexing from the API into Elasticsearch

The domain is recruiter-style job postings with fields such as:
- name
- employmentType
- isRemote
- details
- minSalary
- maxSalary
- location
- companyName
